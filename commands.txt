🚀 ШВИДКИЙ GIT (gpush "message")
  gs          - Статус репозиторію
  gcb         - Створити нову гілку та перейти
  gcam        - Commit -am (додати все + комент)
  gp          - Відправити в хмару (git push)
  gpush "msg" - Повний цикл: add + commit + push однією командою!

📦 РОБОТА З NPM
  run         - Запуск проекту (npm run dev)
  in          - Встановити залежності (npm install)

⚙️ СИСТЕМА ТА ОБСЛУГОВУВАННЯ
  nan         - Редагувати .zshrc (головний конфіг)
  sou         - Оновити налаштування (source)
  update      - Оновити Ubuntu та пакети
  drop        - Очистити RAM кеш (вивільнити пам'ять)
  clean_hist  - Стерти історію команд (чистота!)
  ver         - Перевірити версії Python та Pip
  cls         - Очистити екран термінала

📂 НАВІГАЦІЯ ТА ФАЙЛИ
  win         - Відкрити поточну папку у Windows
  ll          - Список усіх файлів (з прихованими)
  ..          - На рівень вгору
  ~           - Додому (в корінь користувача)

🔍 ШВИДКІ ПОРАДИ (Linux basics)
  grep -r     - Знайти слово всередині файлів
  htop        - Диспетчер завдань (процеси/RAM)
  pwd         - Де я зараз? (шлях до папки)
  mkdir name  - Створити нову папку
  touch file  - Створити пустий файл
  rm -rf      - ВИДАЛИТИ ВСЕ (використовуй з розумом!)
